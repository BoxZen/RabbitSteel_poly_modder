# RabbitSteel_poly_modder

Rabbit & Steel 雙語顯示改裝

### Step-by-Step Guide / 操作步驟

#### 1. Copy Files / 複製檔案
* **EN:** paste `RabbitSteel-poly-modder.exe` into the the game's root directory.
* **CH:** 將下載好的 `RabbitSteel-poly-modder.exe` 貼到遊戲根目錄。
> 📌 **Directory Reference / 路徑參考：**
> `... \Steam\steamapps\common\Rabbit and Steel\`

#### 2. Run the Tool / 執行工具
* **EN:** Double-click `RabbitSteel_poly_modder.exe` to run it. Follow the on-screen prompts and enter the corresponding numbers to choose your languages:
  1. **Sub Language:** The language you want to add as a prefix (e.g., enter `1` for English).
  2. **Main Language:** Your current in-game reading language (e.g., enter `3` for Chinese).
* **CH:** 在該資料夾內，按兩下（雙擊）執行 `RabbitSteel_poly_modder.exe`。請根據畫面提示輸入對應的數字來選擇語言：
  1. **次要語言 (Sub Language)：** 想要加在最前方的語系（例如：輸入 `1` 選擇 English）。
  2. **主要語言 (Main Language)：** 您原本在遊戲中閱讀的語系（例如：輸入 `3` 選擇 Chinese）。

#### 3. Get the Result / 取得成果與套用
* **EN:** Once processing is complete, the tool will automatically generate the bilingual text and create the **`.\Descriptions_Item.csv`** in the same folder.
* **CH:** 處理完成後，程式會自動將合併好的雙語文本創建在同資料夾下的 **`.\Descriptions_Item.csv`**。

### ⚠️ Troubleshooting / 常見問題與排除
* **EN: Will the text overflow the game's UI?**
  Since displaying two languages increases text length, some items with longer descriptions might look slightly crowded or squeezed. This is a visual limitation of the game's original UI.
* **CH: 文字會超出遊戲介面嗎？**
  由於雙語顯示會增加字數，部分道具的描述較長時，遊戲畫面可能會出現微幅的排版擠壓，這屬於遊戲原本 UI 空間限制的正常現象。
